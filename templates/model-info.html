Architecture Details

NeuroVision uses the DeiT smalt architecture, which is based on the Vision Transformer (ViT) design. The model divides input images into fixed-size patches, processes then as a sequence using self-attention mechanisms, and produces classification outputs.

Data Augmentation Pipeline

1. Geometric Transformations

  • Random rotations (±15 degrees)
  • Horizontal and vertical flips
  • Random translations
  • Scaling variations

2. Intensity Jiterations

  • Brightness adjustments (±10%)
  • Contrast modifications
  • Gaussian noise addition (σ = 0.01)

3. GAN-based Synthesis (Will implement this is future to improve the model)

Utilized Generative Adversarial Networks to generate synthetic MRI images with realistic variations.

Preprocessing Pipeline

Image Processing Steps

  1. Normalization: Standardized image intensities to a common scale
  2. Resizing: All images resized to 224×224 pixels to match DeiT model requirements
  3. Channel Adjustment: Converted grayscale images to three-channel format by duplicating the single channel

Training Strategy

K-Fold Cross-Validation

  • Implemented 5-fold cross-validation for robust model assessment
  • Each fold trained independently to ensure generalized feature learning
  • Validation performed on each fold's validation set to monitor overfitting

Training Parameters

Optimizer AdamW

Model Implementation

The model is implemented using PyTorch and the TIMM (PyTorch Image Models) library. Below is the simplified code snippet of the model definition:

import torch.nn as nn
import timm

def create_deit_model(num_classes=4):
    # Load pretrained DeiT small model
    model = timm.create_model('deit_small_patch16_224', pretrained=True)
    
    # Replace the classification head with our own
    model.head = nn.Linear(model.head.in_features, num_classes)
    
    return model

Inference Pipeline

  1. Resize input MRI scan to 224×224 pixels
  2. Apply normalization using ImageNet mean and standard deviation
  3. Pass the processed image through the DeiT model
  4. Apply softmax to get class probabilities
  5. Select the class with the highest probability as prediction
